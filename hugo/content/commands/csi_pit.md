## csi pit

Manipulate or Create a LiveCD (Pre-Install Toolkit)

### Synopsis


Interact with the Pre-Install Toolkit (LiveCD);
create, validate, or re-create a new or old USB stick with
the liveCD tool. Fetches artifacts for deployment.

```
csi pit [flags]
```
